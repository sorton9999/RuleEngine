\doxysubsection{Detailed Description}
Represents a chain of named facts used to track the state and ancestry of rule evaluation in a rule engine. 

A \doxylink{class_rete_core_1_1_token}{Token} encapsulates a set of named facts and maintains a reference to its parent token, forming a linked structure that represents the progression of matched facts in a rule evaluation process. Tokens are typically used to accumulate and access facts as rules are matched and extended. Instances are considered equal if their facts and parent tokens are equal. This class is not thread-\/safe.
