\doxysubsection{Detailed Description}
Represents a terminal node in a Rete network that schedules rule activations when matching facts are asserted. 

A \doxylink{class_rete_core_1_1_terminal_node}{Terminal\+Node} is the endpoint of a Rete rule network. When a fact matching the rule conditions reaches this node, it creates an activation and adds it to the agenda for later execution. \doxylink{class_rete_core_1_1_terminal_node}{Terminal\+Node} instances are typically associated with a specific rule and action. No successor nodes should be added after a \doxylink{class_rete_core_1_1_terminal_node}{Terminal\+Node}.
