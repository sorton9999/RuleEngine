\doxysubsection{Detailed Description}
Provides a fluent interface for constructing and configuring rules in a Rete-\/based rule engine. 

Use the Rule\+Builder to define the sequence of conditions and actions that make up a rule. The builder supports chaining methods to specify matching patterns, logical combinations (AND/\+\+OR), and actions to execute when the rule is triggered. Each method call adds a new node or condition to the rule\textquotesingle{}s execution graph. The builder is not thread-\/safe and should be used from a single thread during rule construction.
